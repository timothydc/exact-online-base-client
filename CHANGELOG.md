# Changelog
All notable changes to this package will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [4.0.0] - 2025-12-14
### Changed
- Namespace

## [2.6.0] - 2022-11-10
### Added
- Extended logging for HTTP requests

## [2.5.2] - 2022-08-15
### Changed
- Allow `mixed` types when setting a Client configuration extension

## [2.5.1] - 2022-08-15
### Changed
- Allow `mixed` return types when getting a Client configuration extension

## [2.5.0] - 2022-08-15
### Added
- Client configuration object can store additional meta data

### Fixed
- Code styling via Bitbucket pipelines

## [2.4.0] - 2022-06-01
### Added
- Optional multi-language functionality in EOL client

## [2.3.1] - 2022-04-12
### Fixed
- Added missing callback getter

## [2.3.0] - 2022-04-12
### Added
- Exact Online language code mapping

## [2.2.0] - 2022-01-19
### Changed
- Rename configuration method

## [2.1.0] - 2022-01-18
### Added
- Support `state` parameter during OAuth

## [2.0.0] - 2021-12-23
### Added
- PHP8.1 support
- more configuration methods
- RateLimit information to RateLimitException

### Removed
PHP7.4 support

## Initial release - 2021-05-12
